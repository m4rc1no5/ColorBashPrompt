Installation
------------

ColorBashPrompt is available at packagist.org, so you can use composer to download this library and all dependencies.

*(add to require section in your composer.json file)*

```json
    {
        "require": {
            "m4rc1no5/color-bash-prompt": "@dev"
        }
    }
```
